# SkillQuest

A Unity-based VR experience project with Oculus/Meta Quest support. 

## Overview

SkillQuest is a virtual reality project built with Unity, designed for Oculus/Meta Quest devices. The project leverages Unity's XR Interaction Toolkit and Input System for immersive VR experiences.

## Tools

- **Unity** - Game engine
- **Unity XR Interaction Toolkit** - VR interactions
- **Unity Input System** - Modern input handling
- **Oculus SDK** - Meta Quest device support

## Prerequisites

- Unity (version specified in ProjectSettings)
- Oculus/Meta Quest headset (for testing)
- Compatible VR development environment

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/TheClassicTechno/skill-quest.git
   cd skill-quest
   ```

2. **Open in Unity**
   - Launch Unity Hub
   - Click "Open" and select the project folder
   - Wait for Unity to import all assets

3. **Configure XR Settings**
   - Navigate to `Edit > Project Settings > XR Plug-in Management`
   - Enable Oculus/Meta Quest support
   - Configure XR settings in `Assets/Settings`

4. **Build and Deploy**
   - Connect your Oculus/Meta Quest device
   - Go to `File > Build Settings`
   - Select Android platform
   - Click "Build and Run"
